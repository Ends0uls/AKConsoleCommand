# knightConsole_Command_site
This is my arkham knight command console site 
features 
1. Copy command to clip board
2. add commands together to make an all in one command
3. toggle on or off each command
4. added a rgb color picker for commands that have rgb
5. added sliders to commands that allow you to change values
6. added search with fuzzy and exact search
7. added Slots for videos
8. added accessibility for accessiblity lol
9. added stats
10. tool tips for what each button does
11. added notes for how to use certain commands that isnt as easy as copy and paste
12. also created template for how to add new commands in the future for me and you ( i forget easly)
